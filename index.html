  1. The exam consists of multiple-choice questions (MCQs).
  2. Semester exams have 40 questions, Final exams have 50 questions.
  3. Each question has only one correct answer.
  4. The exam has a time limit of 60 minutes.
  5. You can navigate between questions using the Previous/Next buttons or the Question Navigator.
  6. Use the "Flag" button to mark questions you want to review later.
  7. Use the "Clear" button to remove your selected answer.

  9. Do not switch tabs/windows during the exam - it may result in automatic submission.

Keyboard Shortcuts:

  • Left Arrow (←): Previous question
  • Right Arrow (→): Next question
  • 1-4: Select option 1 to 4
  • F: Flag current question
  • C: Clear selected answer
